# Assignment 1: Pokémon App - Authentication, Favorites, and Timeline

In this assignment, you are required to continue building your Pokémon App by completing the following epics:

- [Epic 2: User Authentication & Account Management](https://github.com/comp2537-s26/comp2537-s26-course-reader/blob/main/Week-0-Pokemon-App/specs.md#epic-2-user-authentication--account-management)
- [Epic 3: Favorites Management](https://github.com/comp2537-s26/comp2537-s26-course-reader/blob/main/Week-0-Pokemon-App/specs.md#epic-3-favorites-management)
- [Epic 4: User Activity Timeline](https://github.com/comp2537-s26/comp2537-s26-course-reader/blob/main/Week-0-Pokemon-App/specs.md#epic-4-user-activity-timeline)

### Requirements:

- Implement secure login, logout, and registration functionality.
- Support a "Remember Me" option on the login page.
- Allow users to add, view, and remove Pokémons from their favorites list.
- Display a timeline of user activities (login, logout, add/remove favorites) in chronological order, with the ability to delete timeline entries.

Only logged-in users should have access to protected features such as managing favorites and viewing their timeline.

### Notes:

- Passwords must be hashed using `bcrypt`.
- Sessions must be stored persistently.
- User activities must include timestamps and event descriptions.
- Ensure the app behaves gracefully with clear error messages and user-friendly pages.

### Demonstration:

You will demonstrate the working features during lab time. Be ready to show login, logout, registration, adding/removing favorites, and timeline updates.
